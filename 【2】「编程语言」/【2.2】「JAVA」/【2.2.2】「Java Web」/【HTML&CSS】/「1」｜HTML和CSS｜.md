## 创建HTML文件

1. 创建一个web工程（静态的web工程）
2. 在工程下创建html页面

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>标题</title>
</head>
<body>
    hello
</body>
</html>
```
