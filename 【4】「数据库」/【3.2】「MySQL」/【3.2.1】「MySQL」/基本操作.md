 查看服务器中的所有数据库

```mysql> show databases;```

创建新的数据库

```mysql> create database company;```

切换当前工作数据库

```mysql> use company;```

查看当前数据库中的所有表

```
mysql> show tables;
```

查看当前数据库

```show database();```

查看当前用户名

```select user();```

查看服务器状态

```show status;```

查看服务器配置变量

```show variables;```

把.sql文件中数据导入数据库

```source d:\\mywork\\mysql\\company.sql;```
